## 差し戻し事項(1回目)

1. **[重大] ハイ&ロー: 最終カードめくり直後の連打で幽霊カードが混入する**
   - 再現: 最後のプレイヤーがカードをめくった後、結果画面に遷移するまでの約0.9秒間にカードを再タップ
   - 実測: `results` に `{name: "undefined", card: "10♦"}` が追加された。幽霊カードが最弱だった場合「undefinedさんのおごり!」と表示される
   - 原因: 最終めくり後に `isAnimating` が false に戻り、`.is-disabled` はCSSの `cursor` 変更のみでクリックを止めていない
2. **[重大] 全ゲーム共通: アニメーション中に「ゲーム選択に戻る」で戻っても、後から結果画面に強制遷移する**
   - 再現: ルーレット回転中に「← ゲーム選択に戻る」→ ゲーム選択画面に戻れるが、約6秒後に勝手に結果画面へ飛ばされる
   - 原因: 各ゲームのタイマー/アニメーションコールバックが画面離脱後も生きていて `showResult()` を呼ぶ。チンチロ・ハイ&ロー・あみだくじの遅延処理も同構造
